Mailboxes

From this page, you can create, delete, lock and unlock a mailbox. You can also view the size of a mailbox and the number of messages it contains.

Name check box - Select this option to select a mailbox. To select all mailboxes on the server, select the check box at the top of the Name column.

Search - Provides a space for you to type the name of a mailbox that you want to search for. In the drop-down menu, you can select either Name or Locked as a search modifier.

New - Creates a new mailbox in the domain.

Delete - Deletes a mailbox.

Caution

  • If you delete a mailbox, you will also delete all stored e-mail for that mailbox from the mail store.

Lock - Locks a selected mailbox to prevent a user from connecting to the mail server and retrieving e-mail.

Unlock - Unlocks a selected mailbox to allow a user to connect to the mail server and retrieve e-mail.

Note

  • If a domain is locked, you cannot unlock individual mailboxes. To unlock individual mailboxes, you must unlock the domain.
