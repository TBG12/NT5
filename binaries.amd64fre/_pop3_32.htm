Delete a Mailbox

You can delete a mailbox to remove the mailbox from the domain. If you delete a mailbox, you also delete the mail store directory for the mailbox and all e-mail that is stored in that directory. If a user account has been created for the mailbox in the Active Directory or on the local computer, you can also delete it when you delete the mailbox.

To delete a mailbox

  1. On the primary navigation bar, click E-mail.
  2. Click Domains and Mailboxes.
  3. In the Name column, select the check box next to the domain with the mailboxes you want to delete.
  4. Click Mailboxes.
  5. In the Name column, select the check box next to the mailbox you want to delete.
  6. Click Delete, and then click OK.

Notes

  • To perform this procedure, you must be a member of the Administrators group on the local computer, or you must have been delegated the appropriate authority. If the computer is joined to a domain, members of the Domain Admins group might be able to perform this procedure.
  • If you are using Active Directory integrated authentication, you must log on to the Active Directory domain, not the local computer, to perform this procedure.
