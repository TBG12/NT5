E-mail

From this page, you can choose one of the following options to configure the POP3 service:

  • Server Properties - Configure general settings for the POP3 service, such as authentication method, server port, event logging level, Secure Password Authentication (SPA), and mail store directory.
  • Domains and Mailboxes - Manage domains and mailboxes using options such as create, delete, lock, and unlock. You can also access the Mailboxes page from the Domains page.
