Domains

From this page, you can create, delete, lock or unlock a domain. You can also link to the Mailboxes page to manage the mailboxes for the domain you specify.

Name check box - Select this option to select a domain. To select all domains on the server, select the check box at the top of the Name column.

Search - Provides a space for you to type the name of a domain that you want to search for. In the drop-down menu, you can select either Name or Locked as a search modifier.

New - Creates a new domain on the server.

Delete - Deletes a domain.

Caution

  • If you delete an existing domain, you will also delete all stored e-mail for the domain from the mail store.

Mailboxes - Lists the mailboxes for the domain you select. On the Mailboxes page, you can manage individual mailboxes.

Lock - Locks a selected domain to prevent all users in that domain from connecting to the server and retrieving their e-mail.

Unlock - Unlocks a selected domain that has been locked to allow users to connect to the server and retrieve their e-mail.

Note

  • If a domain is locked, you cannot unlock individual mailboxes. To unlock individual mailboxes, you must unlock the domain.
