Add Mailbox

User Name - Provides a space for you to type the name of the mailbox that you want to create.

Password - Provides a space for you to type a password for the new mailbox.

Confirm Password - Provides a space for you to retype the password you entered in Password.

Create associated user for this mailbox - Creates an associated user in Active Directory or on the local computer. You must select this check box if you are using Active Directory integrated authentication or local Windows accounts authentication when creating a mailbox. If the check box is already selected, clear it only if an account already exists with the same name as the mailbox you want to create.
