Lock a Mailbox

You can lock a mailbox to prevent a user from retrieving e-mail. A locked mailbox still receives incoming e-mail, but the user will not be allowed to connect to the server and retrieve e-mail. While a mailbox is locked, the only restriction is that the user cannot connect to the mail server. You can still perform all administrative tasks, such as deleting a mailbox or changing the password for a mailbox.

To lock a mailbox

  1. On the primary navigation bar, click E-mail.
  2. Click Domains and Mailboxes.
  3. In the Name column, select the check box next to the domain that contains the mailbox that you want to lock.
  4. Click Mailboxes.
  5. In the Name column, select the check box next to the mailbox you want to lock.
  6. Click Lock, and then click OK.

Notes

  • To perform this procedure, you must be a member of the Administrators group on the local computer, or you must have been delegated the appropriate authority. If the computer is joined to a domain, members of the Domain Admins group might be able to perform this procedure.
  • If you are using Active Directory integrated authentication, you must log on to the Active Directory domain, not the local computer, to perform this procedure.
