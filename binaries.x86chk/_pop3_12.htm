Server Port

Important

  • If you change this parameter, you must stop and then restart the POP3 service. You can manage the POP3 service from Services on the Administrative Tools menu, which you can access by logging on locally to the server or by connecting to the server using Terminal Services.

The server port is the port that the POP3 service uses to monitor for e-mail clients and to respond to connection requests. Do not change this port from its default of 110, the Internet standard, unless you have a specific need to do so. If you change this port, you must inform all users that you are using a non-standard port because most e-mail clients configure port 110 as the default for connecting to a server running the POP3 service.

To set the server port

  1. On the primary navigation bar, click E-mail.
  2. Click Server Properties.
  3. Type an appropriate value in Server Port, and then click OK.

Note

  • To perform this procedure, you must be a member of the Administrators group on the local computer, or you must have been delegated the appropriate authority. If the computer is joined to a domain, members of the Domain Admins group might be able to perform this procedure.
